# Assets

## Overview

This folder stores supporting project assets used across the AWS Full Observability Platform repository.

Assets may include:

- Project banners
- Icons
- Supporting images
- Visual resources
- Additional media files

---

## Purpose

The goal of this folder is to keep reusable visual resources separated from implementation screenshots and documentation.

This helps maintain a clean project structure and improves repository organization.

---

## Current Usage

Assets can be used in:

- Main README
- Architecture documentation
- Project documentation PDF
- GitHub project previews

---

## Note

Implementation screenshots are stored separately inside:

- monitoring/
- centralized-logging/

to keep workflow-specific visuals organized.
